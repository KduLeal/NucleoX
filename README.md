# NucleoX Landing Page 🚀

Esta é a landing page premium da NucleoX, desenvolvida com HTML5, CSS3 e JavaScript puro. O projeto está otimizado para alta performance e pronto para ser publicado no Vercel.

## 🛠️ Tecnologias Utilizadas
- **HTML5/CSS3** (Vanilla)
- **JavaScript** (Animations & Parallax)
- **Lucide Icons** (Ícones vetoriais)
- **Outfit Font** (Google Fonts)
- **Formspree** (Integração de formulário)

## 🚀 Como Publicar no Vercel

1.  **Prepare o Código:**
    - Abra o arquivo `index.html`.
    - Procure por `YOUR_ID` no formulário de contato e substitua pelo seu ID do [Formspree](https://formspree.io/).
    - (Opcional) Procure por `G-XXXXXXXXXX` e substitua pelo seu ID do Google Analytics.
    - (Opcional) Atualize o link do WhatsApp com o seu número real.

2.  **Upload:**
    - **Opção A (Arrastar):** Acesse [vercel.com](https://vercel.com), faça login e simplesmente arraste a pasta `nucleox-landing` para o dashboard.
    - **Opção B (GitHub):** Suba esta pasta para um repositório no GitHub e conecte o repositório ao Vercel.

3.  **Domínio:**
    - No painel do Vercel, vá em "Settings" > "Domains" para adicionar seu domínio personalizado (ex: `www.nucleox.com.br`).

## 📁 Estrutura de Arquivos
- `index.html`: Estrutura principal e SEO.
- `style.css`: Design system e animações.
- `script.js`: Interatividade.
- `hero-visual.png`: Arte principal gerada por IA.
